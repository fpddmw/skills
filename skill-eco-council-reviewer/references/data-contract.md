# Data Contract

## Ingest JSON
- `event_id`: string
- `generated_at`: UTC ISO string
- `window`: `{start_datetime, end_datetime}`
- `physical_facts`: aggregated snapshot from `physical_metrics`
- `social_opinion`: aggregated snapshot from `social_events`
- `historical_experience`: placeholder block until future skill is linked
- `projection_curve`: placeholder block until future skill is linked

## Enrich JSON
- `provider`: `openai|claude|rule`
- `model`: model id string
- `executive_summary`: string
- `evidence_alignment`: list of `{point,status,rationale}`
- `key_risks`: list of strings
- `key_actions`: list of strings
- `uncertainty_and_gaps`: list of strings
- `confidence`: `{level,reason}`
- `report_markdown`: markdown body (plain string)

## Markdown Brief
Sections generated by `summarize`:
1. 执行摘要
2. 物理事实（Observer）
3. 社会舆论（Listener/Analyzer）
4. 历史经验与推演曲线（当前缺口）
5. 逻辑对齐与共识研判
6. 建议行动
7. 不确定性与数据缺口
8. LLM 原始简报正文
